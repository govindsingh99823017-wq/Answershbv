# Fighter's Classes - Learn Physics Through Experiments

Fighter's Classes is an interactive educational website created by **Govind Singh** (B.Ed + M.Sc. Physics).  
We make physics unforgettable through real experiments, live demonstrations, and over 60 interactive PhET simulations.

## 🌟 Features
- Responsive and modern design  
- Student registration form with email notifications (FormSubmit)  
- Interactive PhET simulations (60+ topics)  
- Downloadable study materials (books, NCERT PDFs)  
- Photo gallery showcasing lab experiments  
- Admin panel to manage books and gallery (password protected)  
- Firebase Firestore backend for dynamic content  
- Fully compatible with GitHub Pages

## 🚀 Deployment on GitHub Pages
1. Upload the entire repository to GitHub.
2. Go to repository Settings → Pages.
3. Set the source branch to `main` (or `master`) and folder to `/ (root)`.
4. Your site will be live at `https://<your-username>.github.io/Fighters-Classes/`.
